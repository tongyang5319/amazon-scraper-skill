# Amazon Scraper Skill

抓取 Amazon 新品榜 / 畅销榜商品信息，支持列表页 + 详情页双重抓取，输出含产品图片的 XLSX 文件。

## 触发命令

`/scrape-amazon`

## 使用方式

```
/scrape-amazon -u <URL> --count <数量>
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-u / --url` | Amazon 类目 URL（必填） | - |
| `--count` | 抓取商品数量 | 50 |
| `--format` | 输出格式：xlsx（含图片）或 csv | xlsx |
| `--postal-code` | 目标国家邮编（如 10001 美国纽约） | - |
| `--retry-asin` | 复查指定 ASIN 的缺失字段 | - |
| `--auto-retry` | 自动复查缺失字段（默认开启） | true |
| `--max-reviews` | 每商品抓取评论数 | 5 |
| `--delay-min` | 详情页最小延迟（秒） | 8 |
| `--delay-max` | 详情页最大延迟（秒） | 15 |

## 功能列表

- 列表页抓取：排名、标题、URL、ASIN、图片、价格、评分、评论数
- 详情页抓取：品牌、月销量、优惠券、产品尺寸/重量、五点描述
- 细分类目排名、A+页面标识
- 用户评论（优先低于3星差评，最多5条）
- 自动滚动加载（Amazon懒加载，100商品全覆盖）
- XLSX 嵌入式产品缩略图
- 默认自动复查缺失字段
- 评星直观显示（⭐⭐⭐⭐⭐）

## 输出文件

`data/新品榜_{类目名}_{日期}.xlsx`

## 常用邮编

| 邮编 | 国家/地区 |
|------|----------|
| 10001 | 美国（New York） |
| 90210 | 美国（Los Angeles） |
| E1 8AN | 英国（London） |
| 10115 | 德国（Berlin） |

## 注意事项

- 详情页请求有8~15秒随机延迟
- 建议每次运行不超过3个类目
- ChromeDriver 已包含在 drivers/ 目录
- 货币由 IP 地理位置决定，与邮编参数无关
